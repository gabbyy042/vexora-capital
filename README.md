## 🎯 Quick Start

### Prerequisites
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web hosting (InfinityFree, Hostinger, etc.)

### Installation

1. **Clone Repository**
   ```bash
   git clone https://github.com/gabbyy042/vexora-capital.git
   cd vexora-capital
   ```

2. **Create Database**
   - Import `database/database_schema.sql` in phpMyAdmin

3. **Configure Application**
   - Copy `config.php.example` → `config.php`
   - Update database credentials & wallet addresses

4. **Upload to Hosting**
   - Upload all files to `public_html`
   - Visit your domain

5. **Test**
   - Register new account
   - Verify email
   - Login to dashboard
   - Admin: gabbyy042 / Chukwu1$
